# Project Adoption PRDs

These local PRDs were synthesized from
`docs/analysis/project-adoption-distribution-plan.md` and follow-up product
decisions from 2026-05-11.

They are intentionally not published to Linear or GitHub. The immediate scope
is local-first project adoption. npm, Homebrew, standalone binary, and desktop
distribution work remains deferred until the local multi-project model is
proven.
