# Issue Tracker: Linear

Issues and PRDs for this repo live in Linear. Use the Linear MCP tools for issue operations.

## Conventions

- Create issues with Linear issue creation tools.
- Read issues with Linear issue lookup/list tools.
- Update issue metadata with Linear issue update tools.
- Add comments with Linear comment tools.
- Apply triage labels using the mappings in `docs/agents/triage-labels.md`.

Use Symphony terminology from `CONTEXT.md` and align work with `SPEC.md`, `SPEC.ext.md`, and the relevant PRD docs under `docs/prd/`.

## When a skill says "publish to the issue tracker"

Create a Linear issue.

## When a skill says "fetch the relevant ticket"

Fetch the Linear issue by identifier, for example `NIE-123`, including description, labels, state, comments, and links when relevant.
